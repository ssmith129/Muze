Borders

Use border utilities to quickly style the border and border-radius of an element. Great for images, buttons, or any other element.

Border

Use border utilities to add or remove an element’s borders. Choose from all borders or one at a time.

Additive

  • Preview
  • HTML 
                    
                      <span class="border"></span>
                      <span class="border-top"></span>
                      <span class="border-right"></span>
                      <span class="border-bottom"></span>
                      <span class="border-left"></span>

Subtractive

  • Preview
  • HTML 
                    
                      <span class="border-0"></span>
                      <span class="border border-top-0"></span>
                      <span class="border border-right-0"></span>
                      <span class="border border-bottom-0"></span>
                      <span class="border border-left-0"></span>

Border color

Change the border color using utilities built on our theme colors.

  • Preview
  • HTML 
                    
                      <span class="border border-primary"></span>
                      <span class="border border-secondary"></span>
                      <span class="border border-success"></span>
                      <span class="border border-danger"></span>
                      <span class="border border-warning"></span>
                      <span class="border border-info"></span>
                      <span class="border border-light"></span>
                      <span class="border border-dark"></span>
                      <span class="border border-white"></span>

Border radius

Add classes to an element to easily round its corners.

  • Preview
  • HTML 
                    
                      <img src="assets/img/svg/placeholder.svg" class="rounded-0" alt="Example non-rounded image (overrides rounding applied elsewhere)" />
                      <img src="assets/img/svg/placeholder.svg" class="rounded" alt="Example 4 rounded image" />
                      <img src="assets/img/svg/placeholder.svg" class="rounded-6" alt="Example 6 rounded image" />
                      <img src="assets/img/svg/placeholder.svg" class="rounded-12" alt="Example 12 rounded image" />
                      <img src="assets/img/svg/placeholder.svg" class="rounded-24" alt="Example 24 rounded image" />
                      <img src="assets/img/svg/placeholder.svg" class="rounded-29" alt="Completely 29 round image" />
                      <img src="assets/img/svg/placeholder.svg" class="rounded-bottom" alt="Example bottom rounded image" />
                      <img src="assets/img/svg/placeholder.svg" class="rounded-left" alt="Example left rounded image" />
                      <img src="assets/img/svg/placeholder.svg" class="rounded-right" alt="Example right rounded image" />
                      <img src="assets/img/svg/placeholder.svg" class="rounded-top" alt="Example top rounded image" />>

Sizes

Use .rounded-1, .rounded-2 or .rounded-3 for larger or smaller border-radius.

  • Preview
  • HTML 
                    
                      <img class="rounded-1" src="..." alt="...">
                      <img class="rounded-2" src="..." alt="...">
                      <img class="rounded-3" src="..." alt="...">
